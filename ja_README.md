# ultra_rm.sh

`ultra_rm.sh` は、カレントディレクトリ内のファイルやディレクトリを一覧表示し、
番号で複数選択して一括削除できるシンプルな対話型シェルスクリプトです。
ディレクトリは中身ごと削除されます。

## 特徴

* カレントディレクトリ内のファイル・ディレクトリを自動で一覧表示
* 複数の項目を番号でまとめて選択可能
* 削除前に確認プロンプトを表示
* ディレクトリは `rm -r` によって再帰的に削除
* POSIX 互換のシンプルな構造

## 使い方

### 1. スクリプトに実行権限を付与

```bash
chmod +x ultra_rm.sh
```

### 2. 実行

```bash
./ultra_rm.sh
```

### 3. 実行例

ディレクトリ内に以下のような内容がある場合:

```
[0] a.txt
[1] memo.md
[2] images
[3] data.json
```

複数選択は次のように入力します:

```
削除したい番号をスペース区切りで入力してください: 1 2
```

すると、スクリプトが選択内容を表示し、確認を求めます:

```
選択されたもの:
 - memo.md
 - images

本当に削除しますか？(y/N):
```

`y` を入力すると削除が実行されます。
それ以外の入力ならキャンセルされます。

## 必要環境

* Bash
* `rm` を扱う覚悟（本当に消えます）

## 注意

このスクリプトは **削除したファイルを復元できません**。
重要なディレクトリでの使用には細心の注意を払ってください。
